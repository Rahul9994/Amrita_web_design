Your Digital Arsenal

Tools & Resources

The essential apps, websites, and platforms every Amrita student needs. Bookmark these and thank us later!

Essential Apps Every CSE Student Should Install

These apps will make your college life 10x easier.

VS Code

The best code editor. Install extensions for Python, C++, Java, and more. Free and lightweight.

Git & GitHub

Version control is non-negotiable. Learn Git early. Host your projects on GitHub to build your portfolio.

Postman

Test APIs like a pro. Essential for web development and backend projects. Free tier is more than enough.

Figma

Design UI/UX for your projects. Free for students. Collaborative and easy to learn. Great for hackathons!

Notion

The ultimate note-taking and productivity app. Organize notes, tasks, projects, and schedules in one place.

Discord

Join coding communities, hackathon servers, and study groups. Most clubs have their own Discord servers too.

IntelliJ IDEA

The best IDE for Java development. Free for students with .edu email. Powerful features for large projects.

Android Studio

Build Android apps. Essential for mobile development enthusiasts. Free and officially supported by Google.

Websites & Platforms for Learning

Level up your skills with these free learning resources.

NPTEL

Free online courses from IIT professors. Amrita has tie-ups with NPTEL. Earn certificates that add value to your resume.

GeeksforGeeks

The bible for CS students. Tutorials on DSA, programming languages, system design, and interview prep. Free and comprehensive.

LeetCode

The #1 platform for coding interview prep. Solve problems daily. Start with Easy, move to Medium, then Hard. Consistency is key!

HackerRank

Practice coding in multiple languages. Great for beginners. Earn badges and certificates. Also has skill certification tests.

freeCodeCamp

Completely free coding curriculum. Web development, data science, machine learning. Build projects and earn certifications.

Coursera

Courses from top universities. Audit for free or apply for financial aid. Great for specialized topics like AI, cloud, and cybersecurity.

YouTube

Channels like Traversy Media, The Net Ninja, freeCodeCamp, and CS50 offer amazing free tutorials. Search and learn!

Stack Overflow

Stuck on a bug? Stack Overflow has the answer. Learn to search effectively and read documentation. Your debugging skills will skyrocket.

Important College Portals

These portals are essential for your daily college life. Bookmark them NOW.

ERP Portal

The central hub for all academic activities. Check your timetable, attendance, marks, fee status, and exam schedules here.

Access via college intranet

Attendance Tracker

Track your attendance percentage subject-wise. Check regularly to ensure you're above 75%. Don't wait till it's too late!

Available on ERP

Digital Library

Access e-books, research papers, journals, and digital resources. Great for project research and assignment references.

Library portal

Placement Portal

Register for placement drives, view job openings, upload your resume, and track your application status. Activate from Year 3.

Placement cell

Free Tools for Projects, Notes & Productivity

Work smarter, not harder.

Google Drive

15GB free cloud storage. Store all your notes, projects, and assignments. Access from anywhere. Share files with teammates easily.

Microsoft 365 (Free for Students)

Get Word, Excel, PowerPoint, and OneDrive free with your .edu email. Essential for assignments, presentations, and reports.

GitHub Student Pack

Free GitHub Pro, domain names, cloud credits, and dozens of developer tools. Apply with your college email. A $1000+ value pack!

Canva

Design posters, presentations, social media posts, and resumes. Free tier is incredibly powerful. No design skills needed!

SmallPDF / ILovePDF

Merge, split, compress, and convert PDFs for free. Essential for submitting assignments and handling official documents.

Trello / Asana

Project management tools for group projects. Create boards, assign tasks, set deadlines, and track progress visually.
